# راهنمای راه‌اندازی آزمون تعیین سطح

## 📋 مراحل راه‌اندازی

### 1. ساخت Migration برای فیلدهای جدید

```bash
cd C:\ctlg\backend
python manage.py makemigrations accounts
python manage.py makemigrations assessment
python manage.py migrate
```

### 2. ایجاد آزمون تعیین سطح با سوالات نمونه

بعد از migration، این دستور را اجرا کنید:

```bash
python manage.py create_placement_test
```

این دستور:
- یک آزمون تعیین سطح اولیه می‌سازد
- 4 سوال نمونه ارزیابی سواد شناختی اضافه می‌کند
- گزینه‌های صحیح و نادرست را تنظیم می‌کند
- امتیازها را تنظیم می‌کند (هر سوال صحیح = 25 امتیاز)

---

## 📝 سوالات نمونه که ایجاد می‌شوند

### سوال 1: واکنش به ادعای فرو ریختن مترو
- **پاسخ صحیح**: بررسی صحت ادعا از طریق کانال‌های رسمی (25 امتیاز)

### سوال 2: شناسایی نشانه‌های اطلاعات نادرست
- **پاسخ صحیح**: استفاده از ادبیات هیجانی بدون ارجاع قابل راستی‌آزمایی (25 امتیاز)

### سوال 3: مدیریت گزارش‌های متناقض
- **پاسخ صحیح**: بررسی کانال‌های رسمی حمل‌ونقل شهری (25 امتیاز)

### سوال 4: راستی‌آزمایی تصویر
- **پاسخ صحیح**: بررسی متادیتای تصویر و تطبیق با اخبار رسمی (25 امتیاز)

## 🎯 نحوه کار سیستم

### برای دانشجویان جدید:
1. دانشجو ثبت‌نام می‌کند (نقش: دانش‌آموز)
2. **به‌صورت خودکار** به صفحه `/placement-test` هدایت می‌شود
3. آزمون تعیین سطح را می‌دهد
4. سطح شناختی اولیه تعیین می‌شود (1 تا 5)
5. بعد از اتمام، به صفحه اصلی هدایت می‌شود

### برای دانشجویان موجود:
- اگر `has_taken_placement_test = False` باشد، هنگام ورود به صفحه اصلی، به آزمون تعیین سطح هدایت می‌شوند

---

## 🔧 مدیریت آزمون تعیین سطح

### مشاهده آزمون تعیین سطح:
- در پنل استاد نمی‌آید (چون `is_placement_test=True`)
- فقط از طریق API قابل دسترسی است: `GET /api/assessment/tests/placement/`

### ویرایش آزمون تعیین سطح:
می‌توانید از Django Admin یا از طریق API ویرایش کنید:

```python
# در Django shell
from assessment.models import CognitiveTest
test = CognitiveTest.objects.get(is_placement_test=True)
# ویرایش کنید...
```

### اضافه کردن سوال بیشتر:
می‌توانید از پنل استاد یا API استفاده کنید، اما باید `is_placement_test=True` را در نظر بگیرید.

## 📊 محاسبه سطح شناختی

بر اساس نمره آزمون:
- **نمره < 20**: سطح 1
- **نمره < 40**: سطح 2
- **نمره < 60**: سطح 3
- **نمره < 80**: سطح 4
- **نمره ≥ 80**: سطح 5

**حداکثر نمره آزمون تعیین سطح**: 100 (4 سوال × 25 امتیاز)

## 🐛 عیب‌یابی

### مشکل: آزمون تعیین سطح یافت نشد
**راه‌حل**: دستور `create_placement_test` را اجرا کنید

### مشکل: دانشجو به آزمون هدایت نمی‌شود
**بررسی کنید**:
1. آیا `has_taken_placement_test = False` است؟
2. آیا آزمون تعیین سطح `is_active=True` است؟
3. آیا API `/api/assessment/tests/placement/` کار می‌کند؟

### مشکل: بعد از آزمون، سطح آپدیت نمی‌شود
**بررسی کنید**:
1. آیا `is_placement_test=True` در آزمون تنظیم شده؟
2. آیا `grade_session` درست کار می‌کند؟
3. لاگ‌های بک‌اند را بررسی کنید
